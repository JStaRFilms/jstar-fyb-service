About This Project

This final year project implements an AI-powered shopping assistant for e-commerce platforms. The system uses natural language processing to understand customer intent and provide relevant product suggestions.

The project integrates OpenAI's GPT-4 for conversation handling, Next.js 14 for the frontend with Server Actions, and Stripe for seamless payment processing.
